OpenSCOW升级：从v1.6.3到v1.6.4

2024年12月30日 · 阅读需 2 分钟
Huangjun
Developer

进入scow部署目录，修改install.yaml，修改imageTag的值，将其改为：v1.6.4

#...
imageTag: v1.6.4
#...

2. 拉取镜像

./cli compose pull mis-web

3. 重启SCOW

./cli compose down
./cli compose up -d

./cli compose ps查看容器状态，当所有容器状态都为UP时表示升级成功。
