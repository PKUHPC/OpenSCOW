# AI 系统简介（Beta）

AI 系统能让您通过 Web 界面管理您在 AI 训练中的数据集、算法、模型，并通过镜像的方式来进行您的 AI 训练任务。

当前 AI 系统仍处于 Beta 版本，使用时如有任何疑问或建议，欢迎在 [Github Issue](https://github.com/PKUHPC/SCOW/issues) 中反馈。

## 功能和特点

- 数据集
  - 个人数据集版本管理
  - 公共数据集分享与使用
- 镜像
  - 生成训练环境镜像
  - 分享和使用公共镜像
- 算法
  - 个人算法版本管理
  - 公共算法分享与使用
- 作业
  - 使用数据集、算法和镜像来快速进行作业训练
  - 作业管理
- 模型
  - 个人模型版本管理
  - 公共模型分享与使用
