| **主机名** | **IP**         | **角色** | **配置** |
| ---------- | -------------- | -------- | -------- |
| manage01   | 192.168.29.106 | 服务节点 | 2C4G     |
| login01    | 192.168.29.101 | 登录节点 | 2C4G     |
| compute01  | 192.168.29.102 | 计算节点 | 2C4G     |
| compute02  | 192.168.29.103 | 计算节点 | 2C4G     |
