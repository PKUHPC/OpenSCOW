认证系统简介

认证系统(auth)为系统提供用户登录功能，以及为各个组件提供鉴权服务。

内置认证系统

我们提供的认证系统实现支持以下认证方式。请根据自己的集群的情况，选择自己的认证方式并进行部署和配置。

  • SSH
  • LDAP
tip

如果您的集群可以使用SSH登录，可以首先使用最简单的SSH方式进行测试。

内置系统的额外配置介绍请参考内置认证系统配置

自定义认证方式

如果您的集群需要使用其他认证方式，您可以参考文档去使用或者实现一个自定义认证系统。
